# CV Tool

Frontend test assignment application for generating cover letters.

## Stack

- Vite
- React
- TypeScript
- pnpm

## Getting Started

```bash
pnpm install
pnpm dev
```

## Available Scripts

```bash
pnpm dev
pnpm build
pnpm lint
pnpm preview
```

## Status

Phase 1 bootstrap is in progress. Tooling, architecture, UI, and deployment setup will be added in the next phases.
